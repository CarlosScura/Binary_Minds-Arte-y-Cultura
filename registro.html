Registrate" → link a registro.html # ================================ # crear_evento.html # ================================ HEREDA de base.html SI no hay sesión activa → redirigir a login FORMULARIO (envía a POST /eventos/crear): - Campo: título (texto, obligatorio) - Campo: descripción (texto largo, obligatorio) - Campo: fecha y hora del evento (fecha, obligatorio) - Campo: ubicación texto (texto, obligatorio) - Campo: embed de Google Maps (texto largo) → instrucción visible: "Pegá acá el iframe de Google Maps" - Botón "Crear evento" SI hay error desde el backend: MOSTRAR mensaje de error en rojo # ================================ # calificar.html # ================================ HEREDA de base.html SI no hay sesión activa → redirigir a login MOSTRAR: - Nombre del usuario a calificar - Nombre del evento FORMULARIO (envía a POST /calificar): - Campo oculto: evaluado_id - Campo oculto: evento_id - Selector de estrellas: 1 a 5 (obligatorio) - Botón "Enviar calificación" SI hay error desde el backend: MOSTRAR mensaje de error en rojo SI ya calificaste a esta persona en este evento: MOSTRAR "Ya calificaste a esta persona en este evento" OCULTAR el formulario
